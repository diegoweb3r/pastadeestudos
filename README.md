O <strong>Pasta de Estudos</strong> é um repositório criado para centralizar materiais de aprendizado, resumos, anotações, exercícios e conteúdos relacionados à área de Tecnologia da Informação. ### 🎯 Objetivo

Este repositório foi desenvolvido para servir como uma biblioteca pessoal de estudos, permitindo registrar conhecimentos adquiridos ao longo da minha formação em Análise e Desenvolvimento de Sistemas e da minha transição para a área de Tecnologia.

Além de auxiliar na revisão de conteúdos, o projeto também funciona como um histórico da minha evolução acadêmica e profissional.

---
